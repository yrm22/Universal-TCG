# 🃏 Universal TCG - Simulador de Sobres

¡Bienvenido al multiverso de cartas! Este es un proyecto de simulador de apertura de sobres y arena de batalla basado en tus series favoritas.

## ✨ Características
* **Sets de Anime:** My Hero Academia, Dragon Ball, Jujutsu Kaisen y más.
* **Sistema de Rarezas:** / Comunes / Poco Comunes / Raras / Super Raras/ Secretas /
* **Economía del Juego:** Gana Koins en la arena y usa Polvos Estelares para mejorar tu colección.
* **Arena de Batalla:** Un tablero de 6x5 donde la estrategia y la suerte de los dados deciden al ganador.

## 🚀 Tecnologías utilizadas
* **HTML5 / CSS3** (con Tailwind CSS)
* **JavaScript** (Lógica pura de juego y persistencia en LocalStorage)

## 🛠️ Instalación y Uso
1. Clona el repositorio.
2. Abre el archivo `index.html` en tu navegador.
3. ¡Empieza a abrir sobres y armar tu mazo!
